#  <img src="https://i.imgur.com/4Kb89tO.png" height="30" /> Authenticator
A simple TOTP plugin for spigot servers.

## Features
- QR Scanning in game. (Custom Map Rendering)
- Command, Grief and Chat Protection.
- Advanced Requests
- Json Storage
- SHA1 Algorithm
